States of Vending Machine:
1. Balance — Display balance amount in machine
2. Show — Show products if already available in machine
3. Refill — Load the machine with the items if the machine is empty
4. Select — Select the item from the available list, enter the price of the item, validate the price and dispense the item on success upon 
successful validation of entered price of the selected item and return the change if applicable 
5. Exit — If the user cancels the transaction, return the amount if inserted in the machine and exit from the system
6. Language — Change language as per user option
